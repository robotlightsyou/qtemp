some basic projector controls

Projector Manager is a free piece of software for controlling Panasonic Prjectors. We can see here that it operates a little differently than QLab. Here there are no properties to modify, just actions to execute. This is because there are no data objects, you're making a physical change to the projector and then the script is done. Using the applescript dictionary and this script as a base, modify until you have scripts that will mute, unmute, power on and power off a projector.
