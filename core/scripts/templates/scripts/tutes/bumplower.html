convert bump up to bump down

Like the disarm script, this is a simple adjustmnet to change the master audio level. This scripts starts by checking the current value of row 0 column 0, which is the master. To adjust other out puts change the column. To adjust the different inputs change the row. Once we have the current value we can add the change amount, and then reassign that value to row 0 column 0. Can you think of how to modify this script so that it lowers the master volume by 2.5db?

tell application id "com.figure53.qlab.4" to tell front workspace
    -- change this number to change the amount to bump by. 
    set changeAmount to 2.5
    set theSelection to selected
    repeat with eachCue in theSelection
        if q type of eachCue is "Audio" or q type of eachCue is "Fade" or q type of eachCue is "Video" then
            tell front workspace
                set currentLevel to eachCue getLevel row 0 column 0
                set newLevel to currentLevel - changeAmount
                eachCue setLevel row 0 column 0 db newLevel
            end tell
        end if
    end repeat
end tell
