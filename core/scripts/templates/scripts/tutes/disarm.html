convert set armed to set disarmed

Here we see armed is a boolean property of a cue object, which means it's only possible states are true and false. Below is a basic script for setting a cue to armed. It starts with the tell block identifying QLab as the intended application focusing on the front window as expected. Next there is a comment from the programmer reiterating the syntax to set values in applescript. Following that we have the declation of the variable myQ.* Recall that green syntax highlighting is for variables. Next in purple we see that we are changing the armed property of myQ to true. And finally we close the script with an end tell.

tell application "com.figurer53.Qlab.4" to tell front workspace
    set myQ to last item of (selected as list)
    set armed of myQ to true 
end tell
